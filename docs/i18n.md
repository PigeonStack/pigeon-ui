# PigeonUI 国际化（i18n）指南

## 概述

PigeonUI 组件库内置了 Qt 标准国际化支持。所有组件内的用户可见文本
均已使用 `qsTr()` 包裹，外部项目可通过 Qt 翻译工具链提取并翻译这些字符串。

## 可翻译字符串清单

| 组件 | 字符串 | 说明 |
|------|--------|------|
| PDialog | `"OK"` | 确认按钮默认文字 |
| PDialog | `"Cancel"` | 取消按钮默认文字 |
| PDialog | `"Dialog"` | 无障碍名称回退 |
| PDrawer | `"Drawer"` | 无障碍名称 |
| PMenu | `"Menu"` | 无障碍名称 |
| PNavBar | `"Navigation"` | 无障碍名称 |
| PList | `"List"` | 无障碍名称 |
| PTabBar | `"Tabs"` | 无障碍名称 |
| PTable | `"Table"` | 无障碍名称 |
| PIconButton | `"Button"` | 无障碍名称回退 |
| PProgressBar | `"Loading"` | 无障碍名称 |
| PSelect | `"Select"` | 无障碍名称回退 |

> 组件公开属性（如 `PDialog.confirmText`、`PButton.text`）由使用者设置，
> 使用者应在自己的代码中使用 `qsTr()` 包裹。

## 翻译工作流

### 1. 提取可翻译字符串

使用 Qt 的 `lupdate` 工具扫描 QML 文件，生成 `.ts` 翻译源文件：

```bash
# 提取 PigeonUI 组件库的字符串
lupdate src/pigeon_ui/PigeonUI/*.qml -ts translations/pigeonui_zh_CN.ts

# 同时提取项目自身的 QML 文件
lupdate yourapp/*.qml src/pigeon_ui/PigeonUI/*.qml -ts translations/app_zh_CN.ts
```

### 2. 翻译 .ts 文件

使用 Qt Linguist 或文本编辑器编辑 `.ts` 文件，为每个 `<source>` 提供 `<translation>`：

```xml
<message>
    <source>OK</source>
    <translation>确定</translation>
</message>
<message>
    <source>Cancel</source>
    <translation>取消</translation>
</message>
```

### 3. 编译翻译文件

使用 `lrelease` 将 `.ts` 编译为二进制 `.qm` 文件：

```bash
lrelease translations/pigeonui_zh_CN.ts
```

### 4. 在应用中加载翻译

在 Python 入口文件中加载 `.qm` 翻译文件：

```python
from PySide6.QtCore import QTranslator, QLocale
from PySide6.QtWidgets import QApplication

app = QApplication([])

translator = QTranslator()
if translator.load(QLocale(), "pigeonui", "_", "translations"):
    app.installTranslator(translator)
```

或在 QML 中通过 `Qt.uiLanguage` 动态切换：

```qml
Component.onCompleted: {
    Qt.uiLanguage = "zh_CN"
}
```

## RTL（从右到左）布局支持

PigeonUI 所有组件均使用标准 Qt Quick 布局模式（`anchors`、`Row`、`Column`），
完全兼容 Qt 的 `LayoutMirroring` 系统。

### 启用 RTL

在应用根级别（如 `Window` 或 `ApplicationWindow`）设置：

```qml
Window {
    LayoutMirroring.enabled: true
    LayoutMirroring.childrenInherit: true

    // PigeonUI 组件会自动镜像布局
}
```

### 自动镜像行为

启用 `LayoutMirroring` 后，以下布局会自动镜像：

- `anchors.left` ↔ `anchors.right` 互换
- `Row` 中子元素顺序反转
- `Text.horizontalAlignment` 方向反转
- `leftPadding` ↔ `rightPadding` 互换

### 不受镜像影响的元素

以下元素保持原始方向（符合预期）：

- **PSwitch** 拇指滑块位置（开关方向与语言无关）
- **PProgressBar** 进度方向（始终从起点到终点）
- **PDrawer** 边缘方向（由 `edge` 属性显式控制）
- **PNavBar/PTabBar** 指示器动画（跟随委托位置）

## 注意事项

- `qsTr()` 中的字符串为英文，作为翻译的 source key。
- 若组件的公开属性已被使用者赋值（如 `confirmText: "好的"`），
  则 `qsTr()` 默认值不会生效，翻译者无需关注。
- 翻译文件建议按语言命名：`pigeonui_<locale>.ts`（如 `pigeonui_ja_JP.ts`）。
