# 架构决策记录（ADR）

本目录存放项目关键架构决策。

## 格式

每个 ADR 文件命名：`NNN-<title>.md`

### 模板

```markdown
# ADR-NNN: 标题

## 状态
已接受 / 草案 / 废弃

## 背景
问题或需求描述。

## 决策
采用的方案。

## 理由
为什么选择该方案。

## 影响
对代码、API、性能的影响。
```
